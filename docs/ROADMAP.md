## 🔄 迭代流程

```
1. 拉取最新代码
   git pull origin main

2. 创建功能分支
   git checkout -b feat/xxx 或 fix/xxx

3. 开发 & 测试
   idf.py build
   idf.py -p COM10 flash

4. 更新交接文档
   - 更新 HANDOFF.md
   - 记录改动和问题

5. 提交代码
   git add .
   git commit -m "feat(xxx): 描述"

6. 合并到 main
   git checkout main
   git merge --no-ff feat/xxx
```
